# Vrinda-Store-Sales-Analysis-Excel
Excel dashboard project analyzing sales, customers, and channels

📌 Project Overview
This project analyzes annual sales data for Vrinda Store using Advanced Excel to uncover customer behavior, top regions, and high-performing sales channels.

🎯 Objectives
Analyze sales trends and order patterns
Identify key customer demographics
Evaluate performance across states and e-commerce platforms
📊 Key Insights
Women contributed ~65% of total sales
Age group 30–49 generated ~50% of revenue
Top states: Maharashtra, Karnataka, Uttar Pradesh
Amazon, Flipkart, and Myntra contributed ~80% of sales
🛠 Tools & Skills Used
Advanced Excel
Pivot Tables & Pivot Charts
Slicers & Interactive Dashboard
Data Cleaning & Processing
📈 Business Impact
The insights from this dashboard can help businesses optimize marketing strategies, focus on high-performing regions, and improve channel-based decision making.
